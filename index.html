It's a simple and easy-to-use Widget component which will search keyword by google search engine.

  • 1.It's simple.Just need to import only one js file which wrote by plain javascript
  • 2.It's customizable.You can simply define some style when calling component or
    completely customize it by yourself via individual CSS file
  • 3.It's asynchronous and won't effect the page performance.
  • 4.It's easy-to-use.You can call it via below way and put the widget into anywhere of page 
                new RecommendWidget({
                keyword: 'android',
                result_size: '3'
            }).populate('widget-area')
